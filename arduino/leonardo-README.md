# Arduino Leonardo

## 基本信息

- **制造商**: Arduino
- **分类**: arduino
- **描述**: 内置USB通信的开发板，可模拟键盘和鼠标

## 硬件规格

- **数字引脚**: 14 个
- **模拟引脚**: 6 个
- **PWM 引脚**: 7 个
- **上传速度**: 57600 bps

## USB 识别

- **VID**: 0x2341
- **PID**: 0x8036

## 核心库

- **核心**: arduino:avr@1.8.6
- **FQBN**: arduino:avr:leonardo

## 所需库

- Servo@1.2.0
- Keyboard
- Mouse

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `leonardo-pins.json` 文件。
