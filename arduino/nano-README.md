# Arduino Nano

## 基本信息

- **制造商**: Arduino
- **分类**: arduino
- **描述**: 小巧紧凑的开发板，适合嵌入式项目

## 硬件规格

- **数字引脚**: 14 个
- **模拟引脚**: 8 个
- **PWM 引脚**: 6 个
- **上传速度**: 57600 bps

## USB 识别

- **VID**: 0x2341
- **PID**: 0x0043

## 核心库

- **核心**: arduino:avr@1.8.6
- **FQBN**: arduino:avr:nano

## 所需库

- Servo@1.2.0

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `nano-pins.json` 文件。
