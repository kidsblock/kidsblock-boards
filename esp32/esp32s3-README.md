# ESP32-S3

## 基本信息

- **制造商**: Espressif
- **分类**: esp32
- **描述**: ESP32的升级版，支持AI加速和USB OTG

## 硬件规格

- **数字引脚**: 45 个
- **模拟引脚**: 20 个
- **PWM 引脚**: 16 个
- **上传速度**: 921600 bps

## USB 识别

- **VID**: 0x303A
- **PID**: 0x1001

## 核心库

- **核心**: esp32:esp32@2.0.14
- **FQBN**: esp32:esp32:esp32s3

## 所需库

- WiFi
- BluetoothSerial
- USB

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `esp32s3-pins.json` 文件。
