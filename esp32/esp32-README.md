# ESP32 DevKit

## 基本信息

- **制造商**: Espressif
- **分类**: esp32
- **描述**: 强大的WiFi和蓝牙双模开发板

## 硬件规格

- **数字引脚**: 40 个
- **模拟引脚**: 18 个
- **PWM 引脚**: 16 个
- **上传速度**: 921600 bps

## USB 识别

- **VID**: 0x10C4
- **PID**: 0xEA60

## 核心库

- **核心**: esp32:esp32@2.0.14
- **FQBN**: esp32:esp32:esp32

## 所需库

- WiFi
- BluetoothSerial

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `esp32-pins.json` 文件。
