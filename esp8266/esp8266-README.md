# ESP8266 NodeMCU

## 基本信息

- **制造商**: Espressif
- **分类**: esp8266
- **描述**: 低成本WiFi开发板，适合物联网项目

## 硬件规格

- **数字引脚**: 17 个
- **模拟引脚**: 1 个
- **PWM 引脚**: 11 个
- **上传速度**: 115200 bps

## USB 识别

- **VID**: 0x1A86
- **PID**: 0x7523

## 核心库

- **核心**: esp8266:esp8266@3.1.2
- **FQBN**: esp8266:esp8266:nodemcu

## 所需库

- ESP8266WiFi

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `esp8266-pins.json` 文件。
