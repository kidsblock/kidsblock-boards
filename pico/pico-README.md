# Raspberry Pi Pico

## 基本信息

- **制造商**: Raspberry Pi
- **分类**: pico
- **描述**: 基于RP2040芯片的高性能开发板

## 硬件规格

- **数字引脚**: 29 个
- **模拟引脚**: 4 个
- **PWM 引脚**: 29 个
- **上传速度**: 115200 bps

## USB 识别

- **VID**: 0x2E8A
- **PID**: 0x0005

## 核心库

- **核心**: rp2040:rp2040@3.6.0
- **FQBN**: rp2040:rp2040:rpipico

## 所需库

无

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `pico-pins.json` 文件。
