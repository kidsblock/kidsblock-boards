# Micro:bit V2

## 基本信息

- **制造商**: BBC
- **分类**: microbit
- **描述**: 教育开发板，内置LED矩阵和传感器

## 硬件规格

- **数字引脚**: 21 个
- **模拟引脚**: 6 个
- **PWM 引脚**: 21 个
- **上传速度**: 115200 bps

## USB 识别

- **VID**: 0x0D28
- **PID**: 0x0204

## 核心库

- **核心**: microbit:nrf52@2.0.0
- **FQBN**: microbit:nrf52:microbitv2

## 所需库

无

## 使用方法

1. 在 KidsBlock 中选择此开发板
2. 系统会自动下载并安装所需的核心库和工具链
3. 连接开发板到电脑
4. 开始编程和上传代码

## 引脚定义

详见 `microbit-pins.json` 文件。
